# Sprint5_Terralab
Sprint 5 referente ao desafio da quarta semana do processo seletivo do Laboratório para Pesquisa e Capacitação em Desenvolvimento de Software - TerraLAB para o Backend.

A aplicação tem por finalidade buscar e armazenar o preço, data e condições de pagamento do smartphone Samsung galaxy s20 FE. Para esta análise foram utilizados os anúncios do site submarino, onde podemos sempre atualizar a busca e o algoritmo retorna se é a melhor hora para comprar o aparelho, ou informa quando foi a melhor oferta.
